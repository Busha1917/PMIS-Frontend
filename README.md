# Admin Template Frontside

Copy this folder into another React + Tailwind project and render `AdminTemplateApp`.

## Requirements

- React
- Tailwind CSS
- lucide-react

## Usage

```tsx
import { AdminTemplateApp } from './admin-template-frontside'

export default function App() {
  return <AdminTemplateApp />
}
```

This is frontend-only. All data is mock data inside the components, so you can replace it with API calls later.
